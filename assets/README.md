# Assets Folder

This folder is for storing images, icons, and other static assets.

## Suggested Structure

```
assets/
├── images/       # Screenshots, illustrations
├── icons/        # SVG icons, favicons
└── fonts/        # Custom fonts (if any)
```

## Usage

- Add your favicon.ico here
- Store any images used in the about section
- Add custom fonts if needed
- Include screenshots for documentation

## Notes

- Optimize images before adding (use WebP format when possible)
- Keep file sizes small for better performance
- Use descriptive filenames (e.g., `book-icon.svg` instead of `icon1.svg`)
